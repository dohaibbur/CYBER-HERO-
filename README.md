# CYBERHERO

**Jeu de Hacking et Cybersécurité**
*Version 0.1.0 Alpha*

## Description
CyberHero est un "serious game" éducatif qui vous plonge dans la peau d'un spécialiste en cybersécurité. À travers une interface de bureau virtuel immersive, vous apprendrez les bases du hacking éthique, de la défense réseau et de l'investigation numérique.

## Fonctionnalités

*   **Bureau Interactif** : Naviguez dans un système d'exploitation simulé avec ses propres applications (Terminal, Navigateur, Email, Paramètres).
*   **Missions Scénarisées** :
    *   **Reconnaissance** : Scannez des réseaux et identifiez les composants.
    *   **Analyse de Paquets** : Interceptez et analysez le trafic réseau (type Wireshark).
    *   **Forensique** : Analysez des fichiers PCAP pour trouver des preuves.
*   **Progression RPG** : Gagnez de l'expérience (XP), montez en niveau et gérez votre réputation et vos crédits.
*   **Outils Réalistes** : Familiarisez-vous avec la ligne de commande et des outils de sécurité simulés.

## Comment Jouer

1.  Lancez l'exécutable `CyberHero.exe`.
2.  Créez un nouveau profil ou chargez une partie.
3.  Consultez votre **Boîte de réception (Email)** pour recevoir vos ordres de mission.
4.  Utilisez le **Terminal** et les applications disponibles pour atteindre vos objectifs.

##  Contrôles

*   **Souris** : Navigation et interaction avec l'interface.
*   **Clavier** : Saisie de texte et commandes dans le terminal.
*   **Échap** : Retour ou fermeture des fenêtres.
